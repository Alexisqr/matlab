# Лабораторна робота № 3
## Мета роботи

Метою даної лабораторної роботи є дослідження методів
відновлення перекручених зображень як за відсутності шумів, так і з ними.

### Завдання 1-2

Завантаження та відображення зображення

У цьому завданні були завантажені тестові зображення з бібліотеки MATLAB
за допомогою функції imread. Для відображення зображень була використана функція imshow. 
Команда figure створює нове вікно для кожного зображення, а функція title додає підпис до нього.

```
% Завдання 1-2
% Завантаження та відображення стандартних зображень
I = imread('cameraman.tif');
figure, imshow(I);
title('Вихідне зображення');
```

### Завдання 3

Здійснити процедуру перекручення зображення змінюючи параметри
LEN і THETA.Відобразити перекручене зображення.

До зображення застосовано фільтр розмиття рухом, який моделює зміщення камери під час зйомки.
Параметри LEN і THETA визначають довжину та напрям розмиття, що дозволяє отримати різні
варіанти перекручених зображень.

```
% Завдання 3-4
% Здійснити процедуру перекручення зображення 
% змінюючи параметри LEN і THETA.
% Відобразити перекручене зображення.

% ЕКСПЕРИМЕНТИ
LEN_values = [10, 30, 50];      % різні довжини змазування
THETA_values = [0, 45, 90];     % різні кути

for i = 1:length(LEN_values)
    for j = 1:length(THETA_values)

        LEN = LEN_values(i);      
        THETA = THETA_values(j);     

        PSF = fspecial('motion', LEN, THETA);

        blurred = imfilter(I, PSF, 'conv', 'circular');
...
```

### Завдання 5

Виконати процедуру відновлення зображення.

Виконано відновлення змазаного зображення методом деконволюції (Wiener-фільтра).

```
% Завдання 5-6
        % Виконати процедуру відновлення зображення.
        restored = deconvwnr(blurred, PSF, 0);
```

### Завдання 7

Виконати зашумлення початкового зображення та повторити пункти 2-6.

До змазаного зображення додано гаусів шум для імітації реальних умов зйомки.
Після цього виконано повторне відновлення з урахуванням шуму,
що дозволяє оцінити вплив шуму на якість результату.

Проведено серію експериментів із різними значеннями параметрів LEN і THETA.

```
...
 % Завдання 7
        % Виконати зашумлення початкового зображення та повторити пункти 2-6.
        noisy = imnoise(blurred, 'gaussian', 0, 0.001);

        restored_noisy = deconvwnr(noisy, PSF, 0.01);

        figure;
        subplot(2,2,1), imshow(I), title('Оригінал');
        subplot(2,2,2), imshow(blurred), title('Змазане');
        subplot(2,2,3), imshow(restored), title('Без шуму');
        subplot(2,2,4), imshow(restored_noisy), title('З шумом');

        sgtitle(['LEN=', num2str(LEN), ', THETA=', num2str(THETA)]);

    end
end
```

## Висновки

У роботі було досліджено перекручення та відновлення зображень. 
Проведені експерименти показали, що зі збільшенням параметра LEN ступінь
розмиття зростає, що ускладнює подальше відновлення. 
Параметр THETA визначає напрям розмиття та впливає на характер спотворення зображення.
Встановлено, що якість відновлення залежить від параметрів розмиття
та наявності шуму. 

## Контрольні запитання

- поясніть процес формування зображення.
  
  Зображення формується так: є оригінальне зображення, яке проходить через систему (наприклад, камеру). Під час цього воно може 
  розмиватися через рух або фокус, а також додається шум. У результаті ми отримуємо вже спотворене зображення.
  
- опишіть модель формування зображень та лінійних перекручень.

  Модель формування описується як згортка оригінального зображення з деяким фільтром (PSF), який задає розмиття.
  Це означає, що кожна точка зображення “розтягується” і впливає на сусідні. Також до результату додається шум.
  Оскільки система вважається лінійною і однаковою для всіх точок, перекручення діють однаково по всьому зображенню.
  
- поясніть процес відновлення перекрученого зображення.
  
  Відновлення - це спроба прибрати розмиття і повернути зображення до початкового вигляду.
  Для цього використовують зворотну операцію до згортки - деконволюцію. На практиці застосовують
  Wiener-фільтр, який дозволяє частково відновити зображення.
  Але якщо є шум, то повністю відновити вже не вдається, і можуть з’являтися спотворення.
