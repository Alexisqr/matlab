# Лабораторна робота № 4
## Мета роботи

Метою даної лабораторної роботи є дослідження застосування
апарату перетворення Фур’є для аналізу спектрів зображень та виконання
фільтрації зображень у частотній області.

### Завдання 1

Завантажте з бібліотеки MATLAB кілька різних зображень.

У цьому завданні були завантажені тестові зображення з бібліотеки MATLAB
за допомогою функції imread. Для відображення зображень була використана функція imshow. 
Команда figure створює нове вікно для кожного зображення, а функція title додає підпис до нього.

```
% Завдання 1
% Завантажте з бібліотеки MATLAB кілька різних зображень.

img1 = imread('cameraman.tif');
img2 = imread('moon.tif');
img3 = imread('coins.png');

figure, imshow(img1), title('Cameraman');
figure, imshow(img2), title('Moon');
figure, imshow(img3), title('Coins');
```
