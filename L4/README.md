# Лабораторна робота № 4
## Мета роботи

Метою даної лабораторної роботи є дослідження застосування
апарату перетворення Фур’є для аналізу спектрів зображень та виконання
фільтрації зображень у частотній області.

### Завдання 1

Завантажте з бібліотеки MATLAB кілька різних зображень.

У цьому завданні були завантажені тестові зображення з бібліотеки MATLAB
за допомогою функції imread. Для відображення зображень була використана функція imshow. 
Команда figure створює нове вікно для кожного зображення, а функція title додає підпис до нього.

```
images = {'cameraman.tif', 'moon.tif', 'coins.png'};
for i = 1:length(images)
    
% Завдання 1: Завантаження
    f = imread(images{i});
    
    if size(f,3) == 3
        f = rgb2gray(f);
    end
    
    figure, imshow(f)
    title(['Завдання 1: Зображення - ', images{i}]);
```

### Завдання 2

З використанням функції F = fft2(f) сформуйте й відобразіть
двовимірні спектри зображень. При відображенні спектра використовуйте або
логарифмічний масштаб, або підберіть масштаб уручну шляхом ділення
значень спектра на деяку константу. Можна, також, пронормувати значення
спектра до максимального, а потім домножити кожне значення на 255
(максимальна яскравість для uint8).

Для кожного зображення було виконано двовимірне дискретне перетворення Фур’є за допомогою функції fft2. У результаті отримано комплексний спектр зображення, який містить інформацію про частотні складові.

Оскільки значення спектра можуть мати великий діапазон, було використано логарифмічне масштабування (log(1 + S)), що дозволяє зробити спектр візуально зрозумілим.


```
    % Завдання 2: Спектр
    F = fft2(f);
    Slog = log(1 + abs(F));
    
    figure, imshow(Slog, [])
    title(['Завдання 2: Спектр - ', images{i}]);
```

### Завдання 5 

З використанням функції fspecial('gaussian') задайте двовимірний

Було створено двовимірний фільтр нижніх частот із гаусівською функцією за допомогою fspecial('gaussian').

```
f = imread('cameraman.tif');
if size(f,3)==3
    f = rgb2gray(f);

[M, N] = size(f);

sigma_values = [1, 5];

for s = 1:length(sigma_values)
    
    sigma = sigma_values(s);
    
    % Завдання 5: Вікно фільтра
    h = fspecial('gaussian', [M N], sigma);
    
    figure, imshow(h, [])
    title(['Завдання 5: Вікно фільтра, sigma = ', num2str(sigma)]);
```

### Завдання 8

Виконайте фільтрацію зображень у частотній області з використанням
фільтрів згідно п.п. 5, 6, 7. Поясніть відмінності в отриманих результатах.

Фільтрація виконувалась відповідно до теореми згортки:

- зображення переводиться у частотну область (fft2),
- спектр множиться на частотну характеристику фільтра,
- результат повертається у просторову область (ifft2).

У результаті:

- високочастотні компоненти пригнічуються,
- зображення стає більш гладким.

```
% Для очищення простору
clear;
close all;
clc;



images = {'cameraman.tif', 'moon.tif'};
sigma = 5;

for i = 1:length(images)
    
    f = imread(images{i});
    
    if size(f,3)==3
        f = rgb2gray(f);
    end
    
    [M, N] = size(f);
    
    % FFT
    F = fft2(f);
    
    % фільтр
    h = fspecial('gaussian', [M N], sigma);
    H = fft2(h);
    
    % Завдання 8: Фільтрація 
    Y = F .* H;
    img_filtered = ifft2(Y);
    
    figure, imshow(f)
    title(['Оригінал - ', images{i}]);
    
    figure, imshow(abs(img_filtered), [])
    title(['Завдання 8: Частотна фільтрація (sigma=', num2str(sigma), ')']);
    
  
end
```

### Завдання 9

З використанням того самого віконного фільтра, заданого функцією
fspecial('gaussian'), виконайте фільтрацію зображення в області
просторових змінних. Порівняйте результат з п.8.

Для порівняння було виконано фільтрацію у просторовій області за допомогою функції imfilter.

Цей метод реалізує згортку зображення з фільтром без переходу у частотну область.

```
f = imread('cameraman.tif');

if size(f,3)==3
    f = rgb2gray(f);
end

% Завдання 9: Просторова фільтрація
h_small = fspecial('gaussian', [15 15], 5);

img_spatial = imfilter(f, h_small);

figure, imshow(f)
title('Оригінал');

figure, imshow(img_spatial, [])
title('Завдання 9: Просторова фільтрація');


```
